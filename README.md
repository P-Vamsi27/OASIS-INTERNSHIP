#  OASIS Internship Machine Learning Projects

This repository contains three ML projects completed as part of the AICTE OASIS Infobyte Internship.

##  Projects
1. **Car Price Prediction**
   - Predicts car prices using features like brand, year, fuel type, and transmission.
   - Built with RandomForestRegressor and deployed via Streamlit.

2. **Iris Flower Classification**
   - Classifies iris species based on sepal and petal measurements.
   - Uses RandomForestClassifier and Streamlit for interactive prediction.

3. **Unemployment Analysis**
   - Predicts unemployment rates based on region, area, and participation rate.
   - Includes data visualization and model deployment.

##  Technologies Used
- Python  
- Pandas, NumPy, Scikit‑Learn  
- Matplotlib, Seaborn  
- Streamlit
